# ThinkLikeAHack3r Resources

Free methodology references for people learning hacking, HackTheBox, TryHackMe, CTFs, and practical penetration testing.

These resources are designed to help beginners move beyond tutorial-following and develop independent black-box reasoning. The goal is not to provide shortcuts or walkthrough answers. The goal is to help you understand what your recon output means, what it implies about the target, and what to test next.

## Current Resources

## Core Principle

> The data is the hint.

Every port, banner, endpoint, error message, redirect, header, permission, and service response is evidence. These resources are meant to help you read that evidence instead of waiting for a walkthrough to tell you what to do.

## Recommended Starting Point

1. Read the [Port Enumeration Glossary](./port-enumeration-glossary.md).
2. Use the [Pentesting Decision Framework](./pentesting-decision-framework.md) when you are stuck.
3. Before running a tool, write a thesis:

```text
Observation:
Interpretation:
Prediction:
Next action:
```
